Here we do CRUD operation on a Course Table. This is done using spring JPA and Spring Hibernate. 
Difference between these approaches:
  ->JDBC 
    lot of SQL query.
    lot of Java code.
  -> Spring JDBC 
    no SQL query.
    less of Java code.
  ->JPA
    no SQL query.
    lesser Java code.(Just map entities)
  ->Spring JPA 
    No SQL
    Enitity will handle the operations
